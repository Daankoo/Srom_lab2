**Тема:** Багаторозрядна модулярна арифметика

**Виконав:** ФІ-33 Буткевич Богдан

**Мова програмування:** С++

**Структура проєкту** (у папці Srom_lab2)

_gcd_lcm.cpp_ — функції найбільшого спільного дільника та найменшого спільного кратного

_function_lab1.cpp_ — просто зібранні до купи усі функції з першої лаби, які використовувалася в цій

_CoreBigInt.cpp_ — базові операції класу BigInt (конструктори, конвертація з/в шістнадцятковий формат, побітові зсуви, методи для роботи з окремими бітами GetBit та SetBit).

_ModAddSub.cpp_ — модульне додавання та віднімання

_ModMulPow.cpp_ — модульне піднесення до степення

_BarrettReduction.cpp_ - реалізація швидкого обчислення за модулем

_main.cpp_ — головний файл програми, який включає зчитування вхідних даних, демонстрацію роботи базових арифметичних функцій, тести на коректність алгоритмів (дистрибутивність) та модулі вимірювання часу виконання (бенчмарки).

_header.h_ — заголовний файл проєкту з оголошенням класу BigInt та всіх необхідних математичних функцій.

_hex1.txt, hex2.txt_ - вхідні данні

_corect.txt_ - перевірка коректності роботи функцій

_result.txt_ - вивід результату роботи функцій

_time.txt_ - вивід замірів часу
